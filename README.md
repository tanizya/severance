# 退職金手取り計算

退職金の手取り額を簡単に計算できるWebアプリです。

https://tanizya.github.io/severance/

## 機能

- 退職金額・勤続年数を入力して税金・手取り額を自動計算
- 退職所得控除、所得税、復興特別所得税、住民税の計算過程を表示
- 障害退職による追加控除に対応
- 税金と手取りの割合を横棒グラフで可視化
- PDF / PNG でレポートをダウンロード

## 計算内容

- 退職所得控除額（勤続20年以下: 40万円×年数 / 20年超: 800万円+70万円×(年数-20)）
- 課税退職所得金額（千円未満切り捨て）
- 所得税（累進税率）
- 復興特別所得税（所得税×2.1%）
- 住民税（10%）

## 技術スタック

- HTML / CSS / JavaScript（単一ファイル構成）
- [html2canvas](https://html2canvas.hertzen.com/) - PNG出力
- [jsPDF](https://github.com/parallax/jsPDF) - PDF出力
- GitHub Pages でホスティング

## 注意事項

本計算はあくまで概算です。正確な税額は税理士または所轄の税務署にご確認ください。
